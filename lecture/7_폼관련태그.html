폼 관련 태그

form 태그는 html 내에서 사용자가 입력할 수 있는 양식을 제공하는 태그이다. form 태그 내의 input 태그들을 통해 사용자가 입력하는 정보를 서버로 넘기는 역할을 한다.

submit 버튼을 눌렀을 때 form 태그에 지정 된 action 속성의 경로로, method 속성에서 지정한 방식으로, form 태그 내부에 있는 input 태그의 값들이 서버로 전송된다. method의 기본 설정은 get 방식으로 되어있고 get 방식의 경우 url에 주소?name=value&name=value와 같이 데이터가 전송 되는 모습을 확인할 수 있다.

fieldset : 폼 요소를 그룹으로 묶는 태그

legend : 묶은 폼 요소에 명칭을 붙이는 태그

필드 셋의 제목을 작성하는 부분

필드 셋으로 묶은 영역별로 구분

text와 관련 된 input 태그

type="text"

한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자

type="password"

비밀번호를 입력할 수 있는 텍스트 상자

type="search", type="url", type="email", type="tel"

겉모습은 텍스트 필드와 비슷하지만 각각의 정보에 맞게 분화된 기능을 제공하는 텍스트 상자





숫자와 관련 된 input 태그

type="number"

입력창에 숫자를 입력하는 것이 기본이지만, 브라우저에 따라 스핀 박스가 표시되기도 한다. (스핀박스 : 입력창 오른쪽 위/아래 화살표 버튼)

type="range"

슬라이드 바를 통해 숫자 지정 가능하다.


날짜/시간 관련한 input 태그

type="date/month/week/time/datetime-local"
